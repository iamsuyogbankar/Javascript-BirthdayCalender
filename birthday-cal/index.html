Helpshift Task — Birthday Cal

What is it?

Birthday Cal is a 7-day calendar which tells you the day (Mon/Tue/Wed...Sun) a person's birthday falls on for a specific year.

How does it work?

Given a list of birthdays and the year as input, the Birthday Cal shows the name of the person under the correct day in the calendar.

Say, you have a list of people: 

 A = 2/15/1978
 B = 5/23/1983

and user selects the year as "2015".

Since A's birthday in 2015 falls on Sunday (because 2/15/2015 is Sunday) and B's birthday falls on Saturday (because 5/23/2015 is Saturday).

On the calendar, you have to put A in Sunday's box. And B in Saturday's box.

Task Details

The birthday list will be entered in a JSON format within textarea. The year will be selected from a dropdown.

You need to process the given JSON input and place people's initials in the card according to these rules:

  • If the box is empty, it should add the day--empty class to the box.
  • Add name as title attribute for each box of a card.
  • Each person must always be rendered as square.
  • Order people inside the card based on their age (youngest to oldest).
  • Each square in a card must be of equal size and fit them in such a way that every person occupies maximum area.

How it works?

  • On initial load calendar should show result for selected year.
  • Year change should update the birthday calendar.
